Falha de Comunicação PLC

📋 Equipamento

  • PLC Siemens S7
  • Rede Ethernet Industrial

⚠ Sintomas

  • PLC não responde ao ping
  • Comunicação intermitente
  • Timeout no supervisório

🔎 Diagnóstico

  1. Verificar link de rede
  2. Testar cabo Ethernet
  3. Checar configuração IP
  4. Analisar logs do switch

🛠 Solução

Foi identificado conflito de IP na rede. Alterado IP do PLC para endereço disponível. Após alteração a comunicação normalizou.

📌 Observações

Recomendado implementar controle de DHCP ou tabela de IP fixa.
